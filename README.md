# elm-preybird
An [ELM](http://elm-lang.org/) language application.

## Getting started

### Install
1. Get the [installer](https://elm-lang.org/install) 
2. Run `elm package install`
3. Run `elm-reactor`
4. Browse [localhost:8000](https://localhost:8000/)
5. Choose an _.elm_-file to browse
